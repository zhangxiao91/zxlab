# ZXLab Visual Guidelines

## Direction

The site should feel restrained, editorial, technical, and personal.

Typography, spacing, and real content should create the visual identity.
Decoration should remain secondary.

## Current decisions

- Use a warm off-white background instead of pure white.
- Use near-black text instead of pure black.
- Prefer borders and spacing over shadows.
- Keep navigation visually quiet.
- Use large editorial headings with tight letter spacing.
- Preserve generous whitespace.
- Use one-column layouts on narrow screens.

## Avoid

- Gradient backgrounds and gradient text
- Glassmorphism
- Decorative floating objects
- Excessive cards
- Oversized marketing slogans
- Icons without semantic purpose
- Animation that does not communicate state
