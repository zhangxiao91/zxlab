# 键盘快捷键清单

这是 v0.1 版，只收录真正高频的操作。你不需要一次背完，先挑每天会用到的三四个，连续用一周就会顺手。

## 先记住这 8 个

| 操作 | Windows | macOS |
| --- | --- | --- |
| 复制 | Ctrl + C | Command + C |
| 粘贴 | Ctrl + V | Command + V |
| 剪切 | Ctrl + X | Command + X |
| 撤销 | Ctrl + Z | Command + Z |
| 保存 | Ctrl + S | Command + S |
| 全选 | Ctrl + A | Command + A |
| 查找 | Ctrl + F | Command + F |
| 切换应用 | Alt + Tab | Command + Tab |

## 写作业和改文档

| 操作 | Windows | 什么时候用 |
| --- | --- | --- |
| 重做 | Ctrl + Y | 撤销多了，想恢复上一步 |
| 加粗 | Ctrl + B | Word、在线文档和大多数编辑器 |
| 新建文档或窗口 | Ctrl + N | 新建一个空白内容 |
| 打开文件 | Ctrl + O | 从软件里选择已有文件 |
| 打印或导出 PDF | Ctrl + P | 提交作业前预览、打印或另存为 PDF |

## 管理文件

| 操作 | Windows | 什么时候用 |
| --- | --- | --- |
| 重命名 | F2 | 给文件换一个清楚的名字 |
| 删除到回收站 | Delete | 普通删除，还能从回收站恢复 |
| 永久删除 | Shift + Delete | 谨慎使用，通常无法从回收站恢复 |
| 新建文件夹 | Ctrl + Shift + N | 整理课程、作业和资料 |
| 打开文件资源管理器 | Win + E | 快速找到下载、文档、U 盘 |

## 浏览网页

| 操作 | Windows | macOS |
| --- | --- | --- |
| 新建标签页 | Ctrl + T | Command + T |
| 关闭标签页 | Ctrl + W | Command + W |
| 恢复刚关闭的标签页 | Ctrl + Shift + T | Command + Shift + T |
| 刷新页面 | Ctrl + R | Command + R |
| 跳到地址栏 | Ctrl + L | Command + L |

## 窗口和截图

- **Win + Shift + S**：框选截图，最推荐先学会。
- **Win + V**：打开剪贴板历史。第一次使用时需要开启。
- **Win + D**：显示桌面，再按一次回到原来的窗口。
- **Win + 方向键**：把窗口贴到屏幕左侧、右侧或最大化。
- **Ctrl + Shift + Esc**：打开任务管理器，软件卡住时很有用。
- macOS 截图可用 **Command + Shift + 4** 框选，或 **Command + Shift + 5** 打开截图工具。

## 一个简单练习

新建一个文档，输入两句话，然后只用键盘完成：全选、复制、粘贴、撤销、保存、查找。最后用截图快捷键截下结果。

别急着追求“全键盘操作”。能少找几次菜单、少点几次鼠标，就已经很值。
