# 安全下载软件指南

这是 v0.1 版，目标很简单：让你装到真正想装的软件，而不是顺手带回一桌面广告。

## 最稳妥的下载顺序

1. 先看 Microsoft Store 或软件自己的应用商店。
2. 没有的话，搜索“软件名 官网”。
3. 在搜索结果中核对域名，不要只看标题和图标。
4. 开源软件优先从项目官网或官方 GitHub 仓库进入下载页。
5. 只有在确认来源可信后，才运行安装包。

## 怎么判断是不是官网

- 域名通常和软件名、公司名一致，不会夹着一长串奇怪单词。
- 官网会有产品介绍、帮助文档、隐私政策或公司信息，不只是一个下载按钮。
- 搜索结果最上面的“广告”不一定是官网，先看域名。
- 如果拿不准，可以先去软件的百科页面、官方社交账号或 GitHub 仓库交叉确认域名。

## 假下载按钮常见长相

- 页面上同时出现多个“立即下载”，颜色特别抢眼，却不写软件版本。
- 按钮旁边写着“高速下载”“安全下载器”“推荐下载”。
- 点击后下载的文件名和软件完全对不上。
- 本来想下载一个播放器，页面却要求先安装“下载助手”。
- 浏览器弹出“电脑已中毒”“驱动过期”等吓人的提示。

遇到这些情况，关掉页面，重新找官网。不要和它斗智斗勇。

## 安装前检查

| 检查项 | 你要看什么 |
| --- | --- |
| 文件名 | 是否和软件名、版本大致一致 |
| 文件类型 | Windows 安装包常见为 exe 或 msi，压缩包常见为 zip |
| 来源 | 是否刚从官网、应用商店或官方仓库下载 |
| 数字签名 | 右键文件，打开“属性”，在有数字签名时核对发布者 |
| 杀毒扫描 | Windows 安全中心是否有警告；拿不准就先别运行 |

不要关闭杀毒软件来迁就一个来路不明的安装包。正规软件一般不会要求你这么做。

## 安装时别一路点“下一步”

- 看清是否勾选了浏览器主页、默认搜索、开机自启动。
- 取消与目标软件无关的“推荐安装”。
- 安装路径不懂可以保留默认值，不必为了省空间乱改系统目录。
- 出现管理员权限确认时，先看发布者和软件名是否正确。
- 如果安装界面突然变成完全无关的软件，直接取消。

## 安装后花一分钟检查

1. 从开始菜单打开软件，确认名称和功能正确。
2. 查看桌面和浏览器有没有多出陌生图标、插件或快捷方式。
3. 打开任务管理器的“启动应用”，看看它是否擅自开机启动。
4. 在“设置 → 应用 → 已安装的应用”里确认没有被带进来的软件。
5. 删除下载目录里不再需要的安装包，但保留重要软件的账号和许可证信息。

## 已经装了奇怪软件怎么办

先断开正在进行的可疑下载，再从“已安装的应用”正常卸载。随后用 Windows 安全中心做一次扫描，检查浏览器扩展和默认搜索设置。若持续弹广告或无法卸载，先截图记录现象，再向熟悉电脑的人求助。

## 防止养蛊：不要把电脑装成工具箱

普通 Windows 用户通常不需要同时安装各种“安全卫士”“软件管家”“驱动管家”“系统清理器”或“加速器”。它们可能重复占用后台、弹出推广、修改默认设置，甚至互相冲突。

Windows 自带的 Windows Security 已经包含 Microsoft Defender Antivirus、Windows Firewall 和基础的病毒威胁防护。日常使用先做好 Windows Update，必要时在 Windows Security 中运行扫描，不要为了安装陌生软件而关闭 Defender 或防火墙。

清理空间时，先处理下载目录、回收站和明确的临时文件，不要随意删除系统目录。学校或单位管理设备、特殊行业软件和企业安全策略可能有额外要求，应以管理员说明为准。
