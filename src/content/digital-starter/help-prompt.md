## 怎么查看设备和系统

Windows 可以按 **Win + I** 打开设置，进入“系统 → 系统信息”查看设备名称、处理器、内存和 Windows 版本。也可以按 **Win + R**，输入 **winver** 查看系统版本。

软件版本通常在“设置”“帮助”或“关于”里。如果是网页问题，请写浏览器名称，例如 Edge、Chrome 或 Firefox。

## 报错不要只说“大概意思”

能复制就复制完整文字；不能复制就截图。报错里的英文、数字和文件路径经常是定位问题的关键，不要只写“它说失败了”。

截图时尽量包含：

- 报错窗口的完整标题和内容。
- 出问题的软件界面。
- 任务栏时间，方便判断发生时间。
- 如果是网页问题，保留地址栏域名，但先遮住账号和隐私参数。

## 一个填写示例

    问题一句话概括：
    下载的 zip 文件双击后提示无法打开。

    设备品牌和型号：
    联想小新 14

    操作系统及版本：
    Windows 11 24H2

    出问题的软件：
    文件资源管理器

    我原本想完成什么：
    解压老师发的课程资料。

    完整报错文字：
    Windows 无法完成提取，目标文件无法创建。

    我已经尝试过：
    重新下载一次；把文件移动到桌面；重启电脑。

    我期望的结果：
    能正常解压并打开里面的 PDF。

这种描述已经足够让别人继续排查文件是否损坏、路径是否过长、磁盘空间是否不足。

## 发出去之前先做隐私检查

遮住身份证号、手机号、学号、家庭住址、账号、二维码、密钥和私人聊天内容。截图里如果有无关窗口，也先裁掉。求助需要信息，但不需要把整台电脑的人生都直播出去。
