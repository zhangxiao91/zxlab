# 文件管理基础

## 先认识几个位置

| 位置 | 常见用途 |
| --- | --- |
| 桌面 | 临时放正在用的东西，不适合长期堆满 |
| 下载 | 浏览器下载的软件、图片、压缩包和 PDF |
| 文档 | 作业、报告、资料和笔记 |
| 图片 | 截图、照片和图片素材 |
| 回收站 | 删除后暂时保留、可以恢复的文件 |
| 此电脑 | 查看磁盘、U 盘和移动硬盘 |

路径只是文件的地址，常见形式如下：

```txt
C:\Users\你的用户名\Downloads
C:\Users\你的用户名\Documents
C:\Users\你的用户名\Desktop
```

## Windows 后缀怎么判断

文件名最后的 `.pdf`、`.docx`、`.zip`、`.exe` 叫扩展名。建议在文件资源管理器中选择“查看 → 显示 → 文件扩展名”，让完整文件名显示出来。

| 后缀 | 大概是什么 | 新手要注意什么 |
| --- | --- | --- |
| `.exe` | 可执行程序 | 双击可能直接运行，不要打开陌生来源的文件 |
| `.msi` | Windows 安装包 | 确认发布者和下载来源 |
| `.zip` / `.rar` / `.7z` | 压缩包 | 先确认来源，再解压内容 |
| `.pdf` | 固定版式文档 | 适合阅读和提交，不代表一定安全 |
| `.docx` / `.xlsx` / `.pptx` | Word、Excel、PowerPoint 文档 | 留意宏、外部链接和受保护视图 |
| `.jpg` / `.png` | 图片 | 图片也可能是伪装，重点看最后后缀 |
| `.mp4` | 视频 | 大文件传输时注意压缩 |
| `.txt` / `.md` / `.html` | 文本、Markdown、网页文件 | 改名不会转换内容格式 |
| `.lnk` | Windows 快捷方式 | 它指向另一个文件，不是文件本体 |
| `.tmp` / `.crdownload` | 临时或未完成的文件 | 确认程序关闭、下载结束后再清理 |

特别注意双后缀：`课程资料.pdf.exe` 最后的真正后缀是 `.exe`。把 `照片.jpg` 改成 `照片.png` 也不会把 JPEG 转成 PNG；格式转换要用“导出”或“另存为”。

## 下载和压缩

下载后先去“下载”文件夹，或用浏览器的 `Ctrl + J` 查看记录。文件很多时，先放进资料目录，再按用途整理。

Windows 可以直接处理 `.zip`。右键压缩包时，优先选择“解压到同名文件夹”，避免文件散落；选中多个文件后可以使用“发送到 → 压缩(zipped)文件夹”。`.rar` 和 `.7z` 需要可信的压缩工具。

大学资料的具体目录结构请看[大学资料文件夹模板](/lab/digital-starter/docs/file-structure)。
