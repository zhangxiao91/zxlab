# 电脑故障排查

## 电脑卡了先看任务管理器

按 `Ctrl + Shift + Esc` 打开任务管理器，先看 CPU、内存、磁盘、网络和启动应用。优先关闭自己打开的软件、游戏和浏览器标签页，不要随便结束不认识的系统进程。

## 磁盘空间不够

先检查下载目录、桌面、回收站、微信或 QQ 缓存，以及不用的大型软件。可以从“设置 → 系统 → 存储”查看占用。不要随便删除 `Windows`、`Program Files` 或看不懂的系统目录。

## 没网时按顺序排查

1. 看 Wi-Fi 是否连接。
2. 试试另一个网站，判断是单个网站还是全部网页打不开。
3. 关闭并重新打开 Wi-Fi，再重启浏览器。
4. 确认校园网是否需要登录。
5. 用手机测试同一个 Wi-Fi。

新手先不要乱改 DNS、代理或网络高级设置。把报错截图和文字保存下来，再使用[电脑问题求助模板](/lab/digital-starter/docs/help-prompt)。
