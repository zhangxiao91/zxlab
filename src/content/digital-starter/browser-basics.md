# 浏览器使用基础

浏览器不只是“上网软件”，也是查资料、上课、提交作业和下载文件的主要入口。先把高频操作用熟，再考虑安装插件。

## 先记住这些操作

| 操作 | Windows 快捷键 |
| --- | --- |
| 新建标签页 | `Ctrl + T` |
| 关闭当前标签页 | `Ctrl + W` |
| 重新打开刚关闭的标签页 | `Ctrl + Shift + T` |
| 打开地址栏 | `Ctrl + L` |
| 页面内查找 | `Ctrl + F` |
| 查看下载记录 | `Ctrl + J` |
| 刷新网页 | `Ctrl + R` |
| 强制刷新 | `Ctrl + F5` |

地址栏既可以输入网址，也可以直接搜索。常用网站可以加入收藏夹，但登录重要账号时要先确认地址栏域名。

## 下载和网页问题

- 下载后先知道文件保存在哪，默认通常是“下载”文件夹。
- 只有一个网站打不开时，可能是网站自身的问题；所有网站都打不开时，再检查 Wi-Fi、校园网和路由器。
- 不要在陌生网站输入密码、验证码或身份证信息。
- 遇到“电脑已中毒”“驱动过期”“立即修复”等弹窗，先关闭页面，不要按提示下载软件。

## 浏览器插件怎么管

插件可以做翻译、广告过滤、截图、密码管理和阅读增强，但它们可能读取你访问的网站内容。

安装前检查：

1. 是否来自浏览器官方扩展商店或产品官网。
2. 权限是否与用途匹配。
3. 是否真的解决了一个明确问题。

不用的插件及时禁用或删除。浏览器插件推荐集中放在[下载中心](/lab/digital-starter/docs/software-toolbox)，本篇只讲使用和判断原则。

## 网站使用习惯

- 重要网站加入收藏，不要每次从陌生搜索结果进入。
- 搜索结果中的广告不一定是官网，先核对域名。
- 论坛和搜索结果适合提供线索，涉及安全、账号、付费和系统设置时回到官方说明。
