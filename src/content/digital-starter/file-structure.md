# 大学资料文件夹模板

这是 v0.1 版使用说明。ZIP 模板文件还没有接入，但你现在就可以照着下面的结构手动创建，大约五分钟。

## 推荐目录结构

    大学资料
    ├─ 00_收件箱
    ├─ 01_课程
    │  ├─ 2026-2027-1
    │  │  ├─ 高等数学
    │  │  │  ├─ 课件
    │  │  │  ├─ 作业
    │  │  │  └─ 复习
    │  │  └─ 大学英语
    │  └─ 2026-2027-2
    ├─ 02_证件与手续
    ├─ 03_项目与比赛
    ├─ 04_个人作品
    ├─ 05_软件与安装包
    └─ 99_归档

## 每个文件夹怎么用

| 文件夹 | 放什么 |
| --- | --- |
| 00_收件箱 | 刚下载、刚收到、暂时没整理的文件 |
| 01_课程 | 按学期和课程存课件、作业、复习资料 |
| 02_证件与手续 | 录取材料、证件照、缴费凭证、申请表 |
| 03_项目与比赛 | 小组项目、社团活动、竞赛资料 |
| 04_个人作品 | 简历、作品集、文章、代码和展示文件 |
| 05_软件与安装包 | 确认安全且可能再次使用的安装包 |
| 99_归档 | 已结束但暂时不能删的课程和项目 |

数字前缀只是为了让重要文件夹固定排序，不喜欢也可以删掉。

## 日常使用方法

1. 浏览器下载的文件先留在“下载”目录，或移动到“00_收件箱”。
2. 每周找十分钟，把收件箱里的文件放到正确位置。
3. 每门课程只保留一个主目录，不要在桌面、微信和网盘各建一套不同版本。
4. 学期结束后，把整学期目录移动到“99_归档”，不要急着删除。

## 文件名也要有规则

推荐把“课程或项目、内容、版本或日期”写进文件名：

    高数作业-第03周-张三.docx
    英语演讲-提纲-v2.docx
    入学材料-缴费凭证-2026-08-20.pdf

尽量避开“新建文档”“作业最终版”“最终版2”这类半年后完全看不懂的名字。

## 证件材料单独注意

- 身份证、银行卡、录取材料等不要放进公开共享文件夹。
- 对外提交前确认是否需要打水印，并遮住无关信息。
- 至少保留一份加密云端或移动硬盘备份。
- 不要把唯一一份重要材料只留在微信聊天记录里。

## 模板 ZIP 状态

可下载的空文件夹 ZIP 仍在制作中，路线页会继续标为“待接入”。正文说明已经可以直接使用，ZIP 接入后会沿用这套结构。
