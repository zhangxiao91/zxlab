# 电脑基础速查表

这份速查表面向刚开始真正使用电脑的准大学生。它不追求把电脑讲得很深，目标是帮你先解决最常见的问题：

* 下载的文件去哪了
* 怎么整理文件夹
* 怎么安装和卸载软件
* 怎么截图、标注和发给别人
* 怎么解压压缩包
* 怎么判断下载页面是否靠谱
* 电脑卡了、没网了、没空间了该先看哪里
* 手机、电脑、平板之间怎么传文件和备份资料

本文主要基于 Windows。macOS 的操作会在少数地方补充，但不作为主线。

---

## 1. 先认识几个最常用的位置

Windows 里，很多新手的问题都来自一句话：**文件不知道放哪了。**

你先记住这几个常见位置：

| 位置  | 常见用途                       |
| --- | -------------------------- |
| 桌面  | 临时放正在用的东西，不建议长期堆满          |
| 下载  | 浏览器下载的软件、图片、压缩包、PDF 通常会在这里 |
| 文档  | 放作业、报告、资料、笔记               |
| 图片  | 截图、照片、图片素材                 |
| 视频  | 本地视频、录屏文件                  |
| 回收站 | 删除后的文件通常先到这里，还可以恢复         |
| 此电脑 | 查看磁盘、U 盘、移动硬盘、下载文件夹等       |

最常见的路径类似这样：

```txt
C:\Users\你的用户名\Downloads
C:\Users\你的用户名\Documents
C:\Users\你的用户名\Desktop
```

其中：

* `C:` 通常是系统盘。
* `Users` 是用户文件夹。
* `Downloads` 是下载文件夹。
* `Documents` 是文档文件夹。
* `Desktop` 是桌面。

看到路径不要慌，它只是文件在电脑里的“地址”。

## 2. 建议先整理一个“大学资料”文件夹

不要把所有东西都放桌面，也不要让“下载”文件夹变成垃圾场。

建议你新建一个总文件夹，例如：

```txt
大学资料
├─ 课程资料
├─ 作业与实验
├─ 证件材料
├─ 项目作品
├─ 临时下载
└─ 归档备份
```

每个文件夹的用途：

| 文件夹   | 用途                 |
| ----- | ------------------ |
| 课程资料  | 老师发的课件、教材、参考资料     |
| 作业与实验 | 作业、实验报告、提交版本       |
| 证件材料  | 录取通知书、身份证扫描件、证件照等  |
| 项目作品  | 网页、小程序、比赛资料、个人作品   |
| 临时下载  | 临时下载的安装包、图片、压缩包    |
| 归档备份  | 已完成、暂时不用但以后可能需要的文件 |

证件材料建议单独保存，不要随便上传到陌生网站，也不要发给陌生人或不可信的 AI 工具。

## 3. 文件名要让未来的自己看懂

糟糕文件名：

```txt
新建文档.docx
作业.docx
最终版.docx
最终最终版.docx
真的最终版.docx
```

更好的文件名：

```txt
高数作业-第1章-张三-2026-07-06.docx
英语演讲稿-v2-2026-07-06.docx
录取材料整理-2026-07.pdf
个人主页初稿-v1.html
```

推荐格式：

```txt
主题-版本-日期
课程名-任务名-姓名-日期
```

常见版本写法：

```txt
v1
v2
draft
final
review
```

少用“最终最终版”，多用日期和版本号。

## 4. 文件扩展名是什么

文件名最后的 `.pdf`、`.docx`、`.zip`、`.exe` 叫扩展名。它告诉电脑这个文件大概是什么类型。

常见扩展名：

| 扩展名             | 类型          | 常见用途      |
| --------------- | ----------- | --------- |
| `.docx`         | Word 文档     | 作业、报告     |
| `.pptx`         | PowerPoint  | 演示文稿      |
| `.xlsx`         | Excel       | 表格        |
| `.pdf`          | PDF         | 阅读、提交、打印  |
| `.zip`          | 压缩包         | 打包多个文件    |
| `.rar`          | 压缩包         | 常见压缩格式    |
| `.7z`           | 压缩包         | 压缩率较高     |
| `.exe`          | Windows 程序  | 安装器或可执行程序 |
| `.msi`          | Windows 安装包 | 软件安装      |
| `.jpg` / `.png` | 图片          | 照片、截图     |
| `.mp4`          | 视频          | 视频文件      |
| `.md`           | Markdown    | 笔记、说明文档   |
| `.html`         | 网页文件        | 简单网页      |

注意：不要随便运行来路不明的 `.exe` 文件。很多恶意软件都会伪装成安装包。

## 5. 下载文件通常在哪里

用浏览器下载文件后，默认通常会进入：

```txt
下载
Downloads
```

你可以这样找：

1. 打开文件资源管理器。
2. 左侧点击“下载”。
3. 按“修改日期”排序，最新下载的文件通常在最上面。

也可以在浏览器里查看下载记录：

* Edge / Chrome：`Ctrl + J`
* 或点击右上角下载图标

如果你刚下载了一个软件，却不知道在哪，先去“下载”文件夹找。

## 6. 安装软件前先确认来源

安装软件时，最重要的是：**尽量从官网或可信应用商店下载。**

优先级大概是：

1. 官网
2. Microsoft Store
3. GitHub 官方 Release 页面
4. 知名软件平台
5. 其他第三方下载站

尽量避免：

* 页面上有很多“高速下载”“普通下载”“安全下载”的下载站
* 点击下载后先给你一个奇怪的小安装器
* 安装过程中默认勾选一堆“推荐软件”
* 软件名看起来像正版，但网址很奇怪
* 下载按钮很多，不知道哪个是真的

下载前可以检查：

* 网站域名是否像官网
* 页面是否有大量广告按钮
* 软件名称是否拼写正常
* 文件扩展名是否合理
* 浏览器是否有安全警告

## 7. 安装包、压缩包和软件本体的区别

很多新手会混淆这三类东西。

| 类型   | 常见形式                    | 意思         |
| ---- | ----------------------- | ---------- |
| 安装包  | `.exe` / `.msi`         | 用来安装软件     |
| 压缩包  | `.zip` / `.rar` / `.7z` | 把多个文件打包在一起 |
| 软件本体 | 开始菜单里的 App / 安装目录里的程序   | 真正运行的软件    |

常见流程：

```txt
下载软件安装包 → 运行安装包 → 安装完成 → 从开始菜单打开软件
```

安装完成后，下载文件夹里的安装包通常可以删除，除非你想留作备份。

## 8. 软件安装时要注意什么

安装软件时，别一路狂点“下一步”。

重点看这几项：

1. 安装路径
   默认装到 `C:\Program Files` 通常没问题。大型游戏或很大的软件可以考虑装到其他盘。

2. 是否创建桌面快捷方式
   常用软件可以创建，不常用软件不用。

3. 是否开机自启动
   除了输入法、网盘、驱动类软件，很多软件没必要开机自启。

4. 是否捆绑安装其他软件
   看到“推荐安装”“附加组件”“浏览器主页”等选项，要仔细看。

5. 是否修改默认浏览器或主页
   不需要就取消。

安装完成后，建议从开始菜单搜索软件名打开，而不是一直点安装包。

## 9. 如何卸载软件

Windows 常见卸载方式：

方法一：

```txt
设置 → 应用 → 已安装的应用 → 找到软件 → 卸载
```

方法二：

```txt
开始菜单 → 搜索软件名 → 右键 → 卸载
```

不建议直接删除安装目录，因为这样可能留下注册表、配置文件和快捷方式。

如果软件卸载后还有残留文件，一般问题不大。新手阶段先别急着用复杂清理工具。

## 10. 压缩和解压怎么做

压缩包就像一个“打包盒”，可以把多个文件装在一起，方便发送和保存。

常见格式：

```txt
.zip
.rar
.7z
```

Windows 可以直接打开 `.zip`，但 `.rar` 和 `.7z` 通常建议安装 7-Zip、Bandizip 等工具。

常见操作：

### 解压

右键压缩包，可以看到类似选项：

```txt
解压到当前文件夹
解压到 xxx\
提取全部
```

推荐选择：

```txt
解压到 xxx\
```

这样会自动创建一个同名文件夹，不容易把文件散得到处都是。

### 压缩

选中多个文件或文件夹，右键选择：

```txt
发送到 → 压缩(zipped)文件夹
```

或者用压缩软件选择：

```txt
添加到压缩包
```

发送作业、图片、项目文件时，如果文件很多，先压缩再发会更清楚。

## 11. 截图怎么做

截图是电脑求助、提交问题、记录信息时最常用的技能之一。

Windows 推荐快捷键：

| 操作      | 快捷键               |
| ------- | ----------------- |
| 区域截图    | `Win + Shift + S` |
| 全屏截图并复制 | `PrtSc`           |
| 当前窗口截图  | `Alt + PrtSc`     |
| 全屏截图并保存 | `Win + PrtSc`     |

最推荐新手记住：

```txt
Win + Shift + S
```

按下后可以框选区域，截图会进入剪贴板。你可以直接粘贴到微信、QQ、Word、PPT 或画图工具里。

如果要保存截图：

1. `Win + Shift + S` 截图。
2. 点击右下角弹出的截图通知。
3. 选择保存。
4. 放到合适文件夹。

截图时尽量注意：

* 把关键报错截全。
* 不要截太小。
* 不要把身份证号、手机号、验证码、密码等敏感信息截进去。
* 求助时最好附上文字说明，不要只甩一张图。

## 12. 浏览器基础

浏览器不只是“上网软件”，它也是大学里最常用的学习工具之一。

推荐先熟悉这些操作：

| 操作          | 快捷键                |
| ----------- | ------------------ |
| 打开新标签页      | `Ctrl + T`         |
| 关闭当前标签页     | `Ctrl + W`         |
| 重新打开刚关掉的标签页 | `Ctrl + Shift + T` |
| 打开地址栏       | `Ctrl + L`         |
| 页面内查找       | `Ctrl + F`         |
| 打开下载记录      | `Ctrl + J`         |
| 刷新网页        | `Ctrl + R`         |
| 强制刷新        | `Ctrl + F5`        |

地址栏既可以输入网址，也可以直接搜索。

例子：

```txt
github.com
VS Code 官网
Windows 截图快捷键
```

建议养成习惯：

* 常用网站加入收藏夹。
* 下载文件后知道去哪里找。
* 插件不要乱装。
* 登录重要账号时看清网址。
* 不要在陌生网站输入密码和验证码。

## 13. 浏览器插件不要装太多

浏览器插件很有用，但插件也有权限风险。

常见插件类型：

* 广告过滤
* 翻译
* 密码管理
* 截图
* 网页剪藏
* 阅读增强

安装插件前看三件事：

1. 来源是否可信。
2. 权限是否过大。
3. 是否真的需要。

如果插件要求读取你访问的所有网站内容，不代表一定有问题，但你要知道它权限很高。

不用的插件建议禁用或删除。

## 15. 电脑卡了先看任务管理器

打开任务管理器：

```txt
Ctrl + Shift + Esc
```

你可以看到：

| 项目   | 意思          |
| ---- | ----------- |
| CPU  | 处理器占用       |
| 内存   | 当前程序占用的运行空间 |
| 磁盘   | 硬盘读写情况      |
| 网络   | 网络使用情况      |
| 启动应用 | 开机自动启动的软件   |

如果电脑很卡，可以先看：

* CPU 是否 100%
* 内存是否快满了
* 是否有奇怪软件占用特别高
* 是否开了太多浏览器标签页
* 是否有大量开机自启动软件

不要随便结束不认识的系统进程。可以优先关闭自己打开的软件、游戏、浏览器标签页。

## 16. 磁盘空间不够怎么办

Windows 常见空间问题来自：

* 下载文件夹堆太多安装包
* 微信 / QQ 文件缓存太大
* 桌面堆了太多视频和压缩包
* 回收站没清空
* 游戏和大型软件装太多
* 系统临时文件太多

可以先做这些：

1. 清理下载文件夹里不用的安装包。
2. 清空回收站。
3. 删除不用的视频和压缩包。
4. 卸载不用的软件。
5. 把大文件移动到其他盘或移动硬盘。
6. 使用 Windows 设置里的“存储”查看空间占用。

路径：

```txt
设置 → 系统 → 存储
```

新手不建议随便删除 `Windows`、`Program Files`、`Users` 下面看不懂的文件夹。

## 17. 没网了先这样排查

遇到网页打不开，先按顺序排查：

1. 看 Wi-Fi 是否连接。
2. 看是不是只有一个网站打不开。
3. 尝试打开另一个网站。
4. 关闭再打开 Wi-Fi。
5. 重启浏览器。
6. 重启电脑。
7. 重启路由器。
8. 检查是不是校园网需要登录。
9. 看手机用同一个 Wi-Fi 能不能上网。

如果只有一个网站打不开，可能是网站问题。
如果所有网站都打不开，可能是网络连接问题。
如果微信能用但网页打不开，可能和 DNS、代理或浏览器设置有关。

新手阶段先不要乱改网络高级设置。先截图报错，再搜索报错关键词。

## 18. 电脑出问题时，怎么描述清楚

向同学、老师、搜索引擎或 AI 求助时，不要只说：

```txt
电脑坏了
软件打不开
这个怎么弄
```

更好的求助模板：

```txt
我正在做：【具体任务】

我使用的设备和系统是：
Windows 11 / Windows 10

我期待的结果是：
【你原本希望发生什么】

实际出现的问题是：
【实际发生了什么】

报错信息是：
【复制报错文字，或者附截图】

我已经尝试过：
1. 【尝试 1】
2. 【尝试 2】

我想知道：
下一步应该怎么排查？
```

例子：

```txt
我正在安装 VS Code。

我使用的是 Windows 11。

我期待安装完成后能从开始菜单打开 VS Code。

实际问题是安装器打开后提示“权限不足”。

我已经尝试过重新下载和重启电脑。

请问下一步应该怎么排查？
```

描述越清楚，别人越容易帮你。

## 19. 临时传文件和长期同步要分开

手机、电脑、平板之间传文件，大概分两种需求。

### 临时传输

适合：

* 发一张截图
* 发一个 PDF
* 传一个安装包
* 手机照片传到电脑

常见方式：

* 微信文件传输助手
* QQ
* AirDrop
* Nearby Share
* 数据线
* 网页版网盘临时上传

注意：微信和 QQ 文件可能会过期，不适合长期保存重要资料。

### 长期同步

适合：

* 课程资料
* 笔记
* 项目文件
* 文档
* 常用资料库

常见方式：

* OneDrive
* iCloud
* Google Drive
* 坚果云
* 百度网盘
* Syncthing
* GitHub

长期同步要注意：

* 不要在多个设备同时改同一个文件。
* 小心同步冲突。
* 小心误删后同步到所有设备。
* 重要资料最好还有一份离线备份。

## 20. 重要资料至少备份一份

重要资料包括：

* 录取通知书
* 身份证扫描件
* 证件照
* 学籍材料
* 课程作业
* 实验报告
* 项目作品
* 简历
* 比赛材料

建议备份策略：

```txt
电脑本地一份
云盘一份
特别重要的资料再放 U 盘或移动硬盘一份
```

简单原则：

> 只有一份的文件，等于随时可能丢。

敏感资料备份时要注意隐私，尽量不要上传到不可信平台。

## 21. 账号安全最少做到这几件事

1. 重要账号不要共用同一个密码。
2. 邮箱、学校账号、支付账号要用强密码。
3. 能开两步验证就开。
4. 验证码不要发给别人。
5. 不要在陌生网站输入账号密码。
6. 不要随便扫陌生二维码。
7. 不要把密码、验证码、身份证号发给 AI。
8. 重要账号绑定的手机号和邮箱要保持可用。

如果你记不住很多密码，可以了解密码管理器，但先确保自己理解它的用途和风险。

## 22. 三个小任务

### 任务 1：整理大学资料文件夹

新建一个总文件夹：

```txt
大学资料
```

里面创建：

```txt
课程资料
作业与实验
证件材料
项目作品
临时下载
归档备份
```

然后把桌面和下载文件夹中能归类的东西放进去。

完成标准：

* 桌面不再堆满临时文件。
* 下载文件夹里没有明显无用的旧安装包。
* 重要资料能在 10 秒内找到。

---

### 任务 2：完成一次截图、压缩和解压

步骤：

1. 用 `Win + Shift + S` 截一张图。
2. 保存到 `大学资料\临时下载`。
3. 右键图片，压缩成 zip。
4. 新建一个文件夹叫 `解压测试`。
5. 把 zip 解压到这个文件夹。
6. 确认图片能正常打开。

完成标准：

* 你知道截图保存在哪里。
* 你能创建 zip 压缩包。
* 你能把压缩包解压到指定位置。

---

### 任务 3：建立常用软件清单

建立一个文档，写下你准备使用的软件。

模板：

```txt
浏览器：
截图工具：
压缩解压：
视频播放：
文档/PDF：
笔记：
同步备份：
密码管理：
编程工具：
```

每一项写：

```txt
软件名：
用途：
下载来源：
是否已安装：
```

完成标准：

* 每类最多选择 1 到 2 个工具。
* 下载来源尽量是官网或可信应用商店。
* 不为了“必装”而装一堆用不到的软件。
