# 电脑基础速查表

这份速查表面向刚开始真正使用电脑的准大学生。它不追求把电脑讲得很深，目标是帮你先解决最常见的问题：

* 下载的文件去哪了
* 怎么整理文件夹
* 怎么安装和卸载软件
* 怎么截图、标注和发给别人
* 怎么解压压缩包
* 怎么判断下载页面是否靠谱
* 电脑卡了、没网了、没空间了该先看哪里
* 手机、电脑、平板之间怎么传文件和备份资料

本文主要基于 Windows。macOS 的操作会在少数地方补充，但不作为主线。

---

## 1. 先认识几个最常用的位置

Windows 里，很多新手的问题都来自一句话：**文件不知道放哪了。**

你先记住这几个常见位置：

| 位置  | 常见用途                       |
| --- | -------------------------- |
| 桌面  | 临时放正在用的东西，不建议长期堆满          |
| 下载  | 浏览器下载的软件、图片、压缩包、PDF 通常会在这里 |
| 文档  | 放作业、报告、资料、笔记               |
| 图片  | 截图、照片、图片素材                 |
| 视频  | 本地视频、录屏文件                  |
| 回收站 | 删除后的文件通常先到这里，还可以恢复         |
| 此电脑 | 查看磁盘、U 盘、移动硬盘、下载文件夹等       |

最常见的路径类似这样：

```txt
C:\Users\你的用户名\Downloads
C:\Users\你的用户名\Documents
C:\Users\你的用户名\Desktop
```

其中：

* `C:` 通常是系统盘。
* `Users` 是用户文件夹。
* `Downloads` 是下载文件夹。
* `Documents` 是文档文件夹。
* `Desktop` 是桌面。

看到路径不要慌，它只是文件在电脑里的“地址”。

## 3. 文件扩展名是什么

文件名最后的 `.pdf`、`.docx`、`.zip`、`.exe` 叫扩展名。它告诉电脑这个文件大概是什么类型。

常见扩展名：

| 扩展名             | 类型          | 常见用途      |
| --------------- | ----------- | --------- |
| `.docx`         | Word 文档     | 作业、报告     |
| `.pptx`         | PowerPoint  | 演示文稿      |
| `.xlsx`         | Excel       | 表格        |
| `.pdf`          | PDF         | 阅读、提交、打印  |
| `.zip`          | 压缩包         | 打包多个文件    |
| `.rar`          | 压缩包         | 常见压缩格式    |
| `.7z`           | 压缩包         | 压缩率较高     |
| `.exe`          | Windows 程序  | 安装器或可执行程序 |
| `.msi`          | Windows 安装包 | 软件安装      |
| `.jpg` / `.png` | 图片          | 照片、截图     |
| `.mp4`          | 视频          | 视频文件      |
| `.md`           | Markdown    | 笔记、说明文档   |
| `.html`         | 网页文件        | 简单网页      |

注意：不要随便运行来路不明的 `.exe` 文件。很多恶意软件都会伪装成安装包。

### Windows 系统常见的后缀问题

Windows 有时会隐藏“已知文件类型的扩展名”。建议打开文件资源管理器，选择“查看 → 显示 → 文件扩展名”，先把完整文件名显示出来。这样你看到的 `报告.pdf` 才是真正的文件名，而不是只显示到 `报告`。

常见后缀可以先这样认：

| 后缀 | 大概是什么 | 新手要注意什么 |
| --- | --- | --- |
| `.exe` | 可执行程序 | 双击可能直接运行，不要打开陌生来源的文件 |
| `.msi` | Windows 安装包 | 通常用于安装软件，也要确认来源 |
| `.zip` / `.rar` / `.7z` | 压缩包 | 先确认来源，再解压其中的内容 |
| `.pdf` | 固定版式文档 | 适合阅读和提交，不代表一定安全 |
| `.docx` / `.xlsx` / `.pptx` | Word、Excel、PowerPoint 文档 | 打开陌生文档时留意宏、外部链接和受保护视图 |
| `.jpg` / `.png` | 图片 | 图片也可能只是伪装，重点看完整后缀 |
| `.mp4` | 视频 | 大文件传输时注意是否被压缩 |
| `.txt` / `.md` / `.html` | 文本、Markdown、网页文件 | 改后缀不会自动把内容转换成另一种格式 |
| `.lnk` | Windows 快捷方式 | 它指向另一个文件，不是文件本体 |
| `.tmp` / `.crdownload` | 临时或尚未完成的下载 | 先确认对应程序已关闭、下载已结束，再决定是否清理 |

特别注意双后缀：`课程资料.pdf.exe` 最后的真正后缀是 `.exe`，不是 `.pdf`。文件图标、文件名中间的文字和“PDF”三个字都不能代替最后的扩展名判断。

改名时只把 `照片.jpg` 改成 `照片.png`，不会把 JPEG 转换成 PNG，只会让文件名和文件内容对不上。需要转换格式时，应该使用软件的“导出”“另存为”或专门的转换功能。

如果一个文件显示为只读、无法编辑或打不开，先确认它的完整后缀、来源和打开它的程序，不要急着反复改后缀。

## 4. 下载文件通常在哪里

用浏览器下载文件后，默认通常会进入：

```txt
下载
Downloads
```

你可以这样找：

1. 打开文件资源管理器。
2. 左侧点击“下载”。
3. 按“修改日期”排序，最新下载的文件通常在最上面。

也可以在浏览器里查看下载记录：

* Edge / Chrome：`Ctrl + J`
* 或点击右上角下载图标

如果你刚下载了一个软件，却不知道在哪，先去“下载”文件夹找。

## 5. 安装软件前先确认来源

安装软件时，最重要的是：**尽量从官网或可信应用商店下载。**

优先级大概是：

1. 官网
2. Microsoft Store
3. GitHub 官方 Release 页面
4. 知名软件平台
5. 其他第三方下载站

尽量避免：

* 页面上有很多“高速下载”“普通下载”“安全下载”的下载站
* 点击下载后先给你一个奇怪的小安装器
* 安装过程中默认勾选一堆“推荐软件”
* 软件名看起来像正版，但网址很奇怪
* 下载按钮很多，不知道哪个是真的

下载前可以检查：

* 网站域名是否像官网
* 页面是否有大量广告按钮
* 软件名称是否拼写正常
* 文件扩展名是否合理
* 浏览器是否有安全警告

## 6. 安装包、压缩包和软件本体的区别

很多新手会混淆这三类东西。

| 类型   | 常见形式                    | 意思         |
| ---- | ----------------------- | ---------- |
| 安装包  | `.exe` / `.msi`         | 用来安装软件     |
| 压缩包  | `.zip` / `.rar` / `.7z` | 把多个文件打包在一起 |
| 软件本体 | 开始菜单里的 App / 安装目录里的程序   | 真正运行的软件    |

常见流程：

```txt
下载软件安装包 → 运行安装包 → 安装完成 → 从开始菜单打开软件
```

安装完成后，下载文件夹里的安装包通常可以删除，除非你想留作备份。

## 7. 软件安装时要注意什么

安装软件时，别一路狂点“下一步”。

重点看这几项：

1. 安装路径
   默认装到 `C:\Program Files` 通常没问题。大型游戏或很大的软件可以考虑装到其他盘。

2. 是否创建桌面快捷方式
   常用软件可以创建，不常用软件不用。

3. 是否开机自启动
   除了输入法、网盘、驱动类软件，很多软件没必要开机自启。

4. 是否捆绑安装其他软件
   看到“推荐安装”“附加组件”“浏览器主页”等选项，要仔细看。

5. 是否修改默认浏览器或主页
   不需要就取消。

安装完成后，建议从开始菜单搜索软件名打开，而不是一直点安装包。

## 8. 如何卸载软件

Windows 常见卸载方式：

方法一：

```txt
设置 → 应用 → 已安装的应用 → 找到软件 → 卸载
```

方法二：

```txt
开始菜单 → 搜索软件名 → 右键 → 卸载
```

不建议直接删除安装目录，因为这样可能留下注册表、配置文件和快捷方式。

如果软件卸载后还有残留文件，一般问题不大。新手阶段先别急着用复杂清理工具。

## 9. 防止养蛊：不要把电脑装成工具箱

普通 Windows 用户不需要同时安装各种“安全卫士”“软件管家”“驱动管家”“系统清理器”或“加速器”。这类软件经常重复占用后台、弹出推广、修改默认设置，甚至互相冲突；为了修一个小问题再装五个工具，最后只会让电脑更难判断。

Windows 自带的 Windows Security 已经包含 Microsoft Defender Antivirus、Windows Firewall 等基础安全能力。对日常上网、写作业、安装常见软件的电脑，先把系统更新做好、保持 Windows Security 正常运行，通常比额外堆叠多个安全软件更合适。

建议养成这几件事：

* 通过“设置 → Windows 更新”及时安装系统更新。
* 在 Windows Security 中查看病毒和威胁防护，需要时运行快速扫描或完整扫描。
* 软件优先从官网、Microsoft Store 或官方扩展商店获取。
* 不要为了安装一个来路不明的软件而关闭 Defender 或防火墙。
* 想清理空间时先清理下载目录、回收站和明确的临时文件，不要随便删除系统目录。

学校或单位统一管理的设备、特殊行业软件和明确要求的企业安全策略可能有额外规定。本节不替代这些管理要求；如果设备属于学校或单位，先遵循管理员的说明。

## 11. 压缩和解压怎么做

压缩包就像一个“打包盒”，可以把多个文件装在一起，方便发送和保存。

常见格式：

```txt
.zip
.rar
.7z
```

Windows 可以直接打开 `.zip`，但 `.rar` 和 `.7z` 通常建议安装 7-Zip、Bandizip 等工具。

常见操作：

### 解压

右键压缩包，可以看到类似选项：

```txt
解压到当前文件夹
解压到 xxx\
提取全部
```

推荐选择：

```txt
解压到 xxx\
```

这样会自动创建一个同名文件夹，不容易把文件散得到处都是。

### 压缩

选中多个文件或文件夹，右键选择：

```txt
发送到 → 压缩(zipped)文件夹
```

或者用压缩软件选择：

```txt
添加到压缩包
```

发送作业、图片、项目文件时，如果文件很多，先压缩再发会更清楚。

## 12. 截图怎么做

截图是电脑求助、提交问题、记录信息时最常用的技能之一。

Windows 推荐快捷键：

| 操作      | 快捷键               |
| ------- | ----------------- |
| 区域截图    | `Win + Shift + S` |
| 全屏截图并复制 | `PrtSc`           |
| 当前窗口截图  | `Alt + PrtSc`     |
| 全屏截图并保存 | `Win + PrtSc`     |

最推荐新手记住：

```txt
Win + Shift + S
```

按下后可以框选区域，截图会进入剪贴板。你可以直接粘贴到微信、QQ、Word、PPT 或画图工具里。

如果要保存截图：

1. `Win + Shift + S` 截图。
2. 点击右下角弹出的截图通知。
3. 选择保存。
4. 放到合适文件夹。

截图时尽量注意：

* 把关键报错截全。
* 不要截太小。
* 不要把身份证号、手机号、验证码、密码等敏感信息截进去。
* 求助时最好附上文字说明，不要只甩一张图。

## 13. 浏览器基础

浏览器不只是“上网软件”，它也是大学里最常用的学习工具之一。

推荐先熟悉这些操作：

| 操作          | 快捷键                |
| ----------- | ------------------ |
| 打开新标签页      | `Ctrl + T`         |
| 关闭当前标签页     | `Ctrl + W`         |
| 重新打开刚关掉的标签页 | `Ctrl + Shift + T` |
| 打开地址栏       | `Ctrl + L`         |
| 页面内查找       | `Ctrl + F`         |
| 打开下载记录      | `Ctrl + J`         |
| 刷新网页        | `Ctrl + R`         |
| 强制刷新        | `Ctrl + F5`        |

地址栏既可以输入网址，也可以直接搜索。

例子：

```txt
github.com
VS Code 官网
Windows 截图快捷键
```

建议养成习惯：

* 常用网站加入收藏夹。
* 下载文件后知道去哪里找。
* 插件不要乱装。
* 登录重要账号时看清网址。
* 不要在陌生网站输入密码和验证码。

## 14. 浏览器插件不要装太多

浏览器插件很有用，但插件也有权限风险。

常见插件类型：

* 广告过滤
* 翻译
* 密码管理
* 截图
* 网页剪藏
* 阅读增强

安装插件前看三件事：

1. 来源是否可信。
2. 权限是否过大。
3. 是否真的需要。

如果插件要求读取你访问的所有网站内容，不代表一定有问题，但你要知道它权限很高。

不用的插件建议禁用或删除。

## 16. 电脑卡了先看任务管理器

打开任务管理器：

```txt
Ctrl + Shift + Esc
```

你可以看到：

| 项目   | 意思          |
| ---- | ----------- |
| CPU  | 处理器占用       |
| 内存   | 当前程序占用的运行空间 |
| 磁盘   | 硬盘读写情况      |
| 网络   | 网络使用情况      |
| 启动应用 | 开机自动启动的软件   |

如果电脑很卡，可以先看：

* CPU 是否 100%
* 内存是否快满了
* 是否有奇怪软件占用特别高
* 是否开了太多浏览器标签页
* 是否有大量开机自启动软件

不要随便结束不认识的系统进程。可以优先关闭自己打开的软件、游戏、浏览器标签页。

## 17. 磁盘空间不够怎么办

Windows 常见空间问题来自：

* 下载文件夹堆太多安装包
* 微信 / QQ 文件缓存太大
* 桌面堆了太多视频和压缩包
* 回收站没清空
* 游戏和大型软件装太多
* 系统临时文件太多

可以先做这些：

1. 清理下载文件夹里不用的安装包。
2. 清空回收站。
3. 删除不用的视频和压缩包。
4. 卸载不用的软件。
5. 把大文件移动到其他盘或移动硬盘。
6. 使用 Windows 设置里的“存储”查看空间占用。

路径：

```txt
设置 → 系统 → 存储
```

新手不建议随便删除 `Windows`、`Program Files`、`Users` 下面看不懂的文件夹。

## 18. 没网了先这样排查

遇到网页打不开，先按顺序排查：

1. 看 Wi-Fi 是否连接。
2. 看是不是只有一个网站打不开。
3. 尝试打开另一个网站。
4. 关闭再打开 Wi-Fi。
5. 重启浏览器。
6. 重启电脑。
7. 重启路由器。
8. 检查是不是校园网需要登录。
9. 看手机用同一个 Wi-Fi 能不能上网。

如果只有一个网站打不开，可能是网站问题。
如果所有网站都打不开，可能是网络连接问题。
如果微信能用但网页打不开，可能和 DNS、代理或浏览器设置有关。

新手阶段先不要乱改网络高级设置。先截图报错，再搜索报错关键词。

## 19. 电脑出问题时，怎么描述清楚

向同学、老师、搜索引擎或 AI 求助时，不要只说：

```txt
电脑坏了
软件打不开
这个怎么弄
```

更好的求助模板：

```txt
我正在做：【具体任务】

我使用的设备和系统是：
Windows 11 / Windows 10

我期待的结果是：
【你原本希望发生什么】

实际出现的问题是：
【实际发生了什么】

报错信息是：
【复制报错文字，或者附截图】

我已经尝试过：
1. 【尝试 1】
2. 【尝试 2】

我想知道：
下一步应该怎么排查？
```

例子：

```txt
我正在安装 VS Code。

我使用的是 Windows 11。

我期待安装完成后能从开始菜单打开 VS Code。

实际问题是安装器打开后提示“权限不足”。

我已经尝试过重新下载和重启电脑。

请问下一步应该怎么排查？
```

描述越清楚，别人越容易帮你。

## 20. 临时传文件和长期同步要分开

手机、电脑、平板之间传文件，大概分两种需求。

### 临时传输

适合：

* 发一张截图
* 发一个 PDF
* 传一个安装包
* 手机照片传到电脑

常见方式：

* 微信文件传输助手
* QQ
* AirDrop
* Nearby Share
* 数据线
* 网页版网盘临时上传

#### 微信传文件时先分清“图片”和“文件”

微信聊天里直接发送图片、视频时，软件可能会为了传输速度和体积压缩内容。需要保留原始质量时，在选择图片或视频后勾选“原图”或“原视频”；即使这样，聊天发送也更适合临时分享，不要把它当成原始素材的唯一备份。

如果要传 Word、Excel、PPT、PDF、ZIP 或安装包，优先使用“文件”入口发送，不要先截图、转成图片，或把文件拖进图片区域。文件方式更容易保留原始文件名、扩展名和内容。

微信文件传输助手适合把手机上的文件临时转到电脑，或在自己的设备之间中转。它不是资料库：聊天文件可能过期，微信缓存也可能被清理，文件在不同设备上的保存位置也不一定相同。

#### 微信文件为什么会显示只读

从微信聊天记录或微信缓存目录直接打开的文件，可能被系统或应用以只读方式打开。Word 还可能因为来源不明而进入“受保护的视图”。这不一定表示发送者给文件设置了权限，常见情况是文件仍停留在微信的临时目录，或者 Office 在保护状态下打开了它。

稳妥的做法是：

1. 先把文件“另存为”或复制到自己的“文档”“课程资料”等目录。
2. 确认文件来源可信后，再在 Word 或 PDF 软件里启用编辑。
3. 如果仍然只读，右键文件 → 属性，检查“只读”是否被勾选；同时确认文件没有被其他程序占用。
4. 编辑完成后，把正式版本保存到自己的资料目录，不要只留在微信聊天或缓存里。

注意：微信传输文件可能会过期，不适合长期保存重要资料；重要文件应该放在正式资料目录，并至少保留一份独立备份。

### 长期同步

适合：

* 课程资料
* 笔记
* 项目文件
* 文档
* 常用资料库

常见方式：

* OneDrive
* iCloud
* Google Drive
* 坚果云
* 百度网盘
* Syncthing
* GitHub

长期同步要注意：

* 不要在多个设备同时改同一个文件。
* 小心同步冲突。
* 小心误删后同步到所有设备。
* 重要资料最好还有一份离线备份。

## 21. 重要资料至少备份一份

重要资料包括：

* 录取通知书
* 身份证扫描件
* 证件照
* 学籍材料
* 课程作业
* 实验报告
* 项目作品
* 简历
* 比赛材料

建议备份策略：

```txt
电脑本地一份
云盘一份
特别重要的资料再放 U 盘或移动硬盘一份
```

简单原则：

> 只有一份的文件，等于随时可能丢。

敏感资料备份时要注意隐私，尽量不要上传到不可信平台。

## 22. 账号安全最少做到这几件事

1. 重要账号不要共用同一个密码。
2. 邮箱、学校账号、支付账号要用强密码。
3. 能开两步验证就开。
4. 重要账号绑定的手机号和邮箱要保持可用。

如果你记不住很多密码，可以了解密码管理器，但先确保自己理解它的用途和风险。

## 23. 反诈与防诈骗

电脑和手机上的诈骗，通常不是“技术太高”，而是利用你着急、害怕或想占便宜的那几分钟。遇到下面这些情况，先停下来，不要立即点击、转账或授权：

* 对方催你提供验证码、密码、身份证号、银行卡信息或屏幕共享。
* 对方声称自己是客服、老师、学校、平台工作人员，并要求你马上操作。
* 对方发来“退款”“取消会员”“账号异常”“中奖”“兼职刷单”“高薪实习”等链接。
* 网页要求你下载所谓的安全控件、远程控制软件或“反诈验证工具”。
* 下载页面出现“电脑已中毒”“驱动过期”“立即修复”等吓人的弹窗。
* 对方要求你绕过 Windows 安全提示、关闭 Defender，或从不明网盘安装软件。

可以记住一个简单顺序：**不急着回、不点陌生链接、不提供验证码、不向陌生账户转账、不安装对方指定的远程软件。**

判断链接和软件时，自己打开官方 App、官网或学校公布的渠道，不要直接相信对方发来的地址。下载软件前可以先阅读[安全下载软件指南](/lab/digital-starter/docs/safe-download)。

如果已经点开链接或提供了信息：

1. 先停止继续操作，不要因为对方催促而补交更多信息。
2. 立刻通过官方渠道修改相关密码，并检查账号登录记录。
3. 如果泄露了验证码、支付信息或已经转账，尽快联系银行、支付平台和当地警方。
4. 保留聊天记录、网址、转账凭证和截图，不要只凭记忆描述。

这部分不能替代学校、银行、平台或警方的正式通知；它的作用是帮你先停一下，给自己留出核验时间。

## 24. 三个小任务

### 任务 1：整理大学资料文件夹

新建一个总文件夹：

```txt
大学资料
```

里面创建：

```txt
课程资料
作业与实验
证件材料
项目作品
临时下载
归档备份
```

然后把桌面和下载文件夹中能归类的东西放进去。

完成标准：

* 桌面不再堆满临时文件。
* 下载文件夹里没有明显无用的旧安装包。
* 重要资料能在 10 秒内找到。

---

### 任务 2：完成一次截图、压缩和解压

步骤：

1. 用 `Win + Shift + S` 截一张图。
2. 保存到 `大学资料\临时下载`。
3. 右键图片，压缩成 zip。
4. 新建一个文件夹叫 `解压测试`。
5. 把 zip 解压到这个文件夹。
6. 确认图片能正常打开。

完成标准：

* 你知道截图保存在哪里。
* 你能创建 zip 压缩包。
* 你能把压缩包解压到指定位置。

---

### 任务 3：建立常用软件清单

建立一个文档，写下你准备使用的软件。

模板：

```txt
浏览器：
截图工具：
压缩解压：
视频播放：
文档/PDF：
笔记：
同步备份：
密码管理：
编程工具：
```

每一项写：

```txt
软件名：
用途：
下载来源：
是否已安装：
```

完成标准：

* 每类最多选择 1 到 2 个工具。
* 下载来源尽量是官网或可信应用商店。
* 不为了“必装”而装一堆用不到的软件。
